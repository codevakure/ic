# Ranger Chat Screen Interface

Welcome to Ranger! This comprehensive guide will help you navigate the interface and understand all the key features available to make your experience productive and efficient.

## Getting Started

![Chat Screen Interface](/images/screen.png)

### Home Screen Overview

The Home Screen is your central hub in Ranger, designed to help you start tasks quickly and intuitively.

#### Model Selection (Top-Left Corner)

Choose the Ranger model that best fits your task requirements:

- **Instant Model**: Best for fast, simple responses, quick questions, and casual chats
- **Thinking/Coding Model**: Ideal for balanced reasoning, programming tasks, and technical explanations
- **Research Model**: Perfect for deep analysis, complex tasks, and multi-step reasoning

<Callout type="tip">
  If you're unsure which model to choose, start with the Thinking/Coding Model—it's versatile and handles most tasks well.
</Callout>

## Interface Navigation

### Left Navigation Panel

The Left Navigation Panel helps you manage conversations, collaborate, and access essential tools.

#### Core Navigation Features

**New Chat**: Start a fresh conversation for new projects or topics to avoid confusion with previous threads.

**Search Bar**: Locate specific messages or topics across all your chats using keywords like "report," "audit," or "policy."

**Bookmarks**: Save important messages for quick access later. All bookmarks are stored in one organized location.

**My Agents**: Access and manage specialized agents customized for different tasks, departments, or workflows.

#### Chat Management

**Chat History**: View and manage all conversations organized by date (Today's Chats and Previous Days).

Available actions for each chat:
- **Rename**: Label chats for clarity (e.g., "Q3 Risk Analysis")
- **Duplicate**: Create copies for similar tasks or templates
- **Delete**: Remove outdated or irrelevant conversations

**Close Sidebar Icon**: Hide the sidebar to maximize workspace when focusing on tasks.

#### User Profile

Access personal file management and account actions through your User Profile (bottom left):

- **Files**: View and manage uploaded documents
- **Help & Documentation**: Access user guides
- **Log Out**: Securely exit your account

## Chat Interface Features

### Message Interaction

#### Composing Messages

**Message Box**: Located at the bottom of the screen, type your requests in plain language. No special commands needed—Ranger understands context and responds with relevant insights.

**Microphone**: Click the microphone icon to speak your request for hands-free interaction during meetings or multitasking.

#### File Attachment Options

**Attach File**: Upload documents to enhance your conversations. Choose from two upload modes:

1. **Upload Image**: For visual content like scanned documents, charts, or infographics
2. **Upload Documents**: Enables advanced document search with automatic File Search Tool activation

**Supported File Formats**: Microsoft Office (.docx, .pptx, .xlsx), PDF, CSV, text files, HTML, JSON, YAML, SQL, and various source code formats.

#### Response Interaction Options

When you receive a response from Ranger, you'll see several action options below each message:

- **Read Aloud**: Listen to the response using text-to-speech
- **Copy**: Copy the response content to your clipboard
- **Edit**: Modify the AI's response to better suit your needs
- **Fork**: Create a new conversation branch from this point
- **Like/Dislike**: Provide feedback to improve response quality
- **Regenerate**: Request a new response to the same query

<Callout type="info">
  These response options appear below every AI message, giving you full control over how you interact with and use the generated content.
</Callout>

### Tools Panel

Access specialized features that enhance your workflow. Most tools are automatically activated based on your prompts, while Web Search and Connectors require manual selection from the Tools panel.

#### Available Tools

**Web Search**: Perform real-time searches for current information, market trends, or regulatory updates. **Requires manual selection from Tools panel.**

**Connectors**: Access Microsoft 365 services (SharePoint, OneDrive, Outlook, Teams). **Requires manual selection from Tools panel.**

**Tool Management**: Click on any active tool name to unselect and deactivate it, keeping your workspace clean and focused.

### Top-Right Features

The top-right corner provides conversation management tools that vary based on your current state:

#### New Chat Options

**Temporary Chat**: Start quick, unsaved sessions for testing, short questions, or sensitive information.

**Theme Toggle**: Switch between light and dark themes to match your preference and working environment.
