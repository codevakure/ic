# Fetch URL

The Fetch URL tool allows Ranger to retrieve and analyze content from specific web pages. This tool is automatically activated when you provide a URL in your prompt and ask questions about that webpage.

<Callout type="info">
  Fetch URL activates automatically when you provide a URL and ask questions about the webpage content. No manual tool selection is needed.
</Callout>

## How to Use Fetch URL

Simply provide a URL along with your question about the webpage content in your message. Ranger will automatically detect the URL and activate Fetch URL to retrieve and analyze the content.

**Format:** Provide the URL followed by your specific question about the webpage content.

![Fetch URL Example](/images/fetchurl.png)

**Example:** "Can you fetch this URL https://www.sec.gov/search-filings and provide information about recent SEC filings?"

Ranger will:
1. Automatically detect the URL in your message
2. Access the specified webpage
3. Retrieve the relevant content
4. Extract the requested information
5. Provide a comprehensive response based on the fetched webpage data

## Key Features

- **Direct URL Access:** Fetch content from any publicly accessible web page
- **Targeted Queries:** Ask specific questions about the fetched webpage content
- **Webpage Analysis:** Analyze news articles, company pages, research papers, and more
- **Real-time Data:** Access the most current information available on the web
- **Comprehensive Responses:** Get detailed analysis and extracted information

## Example Use Cases

- Extract information from company websites
- Research current news articles and press releases
- Review online documentation and specifications
- Access and summarize research papers or articles
- Gather data from government websites
- Analyze public web pages and online resources

## Best Practices

- **Be Specific:** Clearly state what information you want from the webpage
- **Use Public URLs:** Ensure the URL is publicly accessible
- **Combine Queries:** You can ask multiple questions about the same webpage content
- **Verify Sources:** Always verify the credibility of the source URL

<Callout type="tip">
  When using Fetch URL, be as specific as possible about what information you need from the webpage. This helps Ranger provide more targeted and relevant responses.
</Callout>

<Callout type="info">
  Fetch URL works best with publicly accessible web pages. Some websites may have restrictions that prevent content fetching.
</Callout>
