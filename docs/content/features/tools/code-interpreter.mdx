# Code Interpreter

The Code Interpreter tool is automatically activated when you request to generate professional documents and presentations. This powerful tool creates various types of content in Texas Capital Bank's branded template format.

<Callout type="info">
  Code Interpreter activates automatically when you request document or presentation generation. No manual tool selection is needed.
</Callout>

## What is Code Interpreter?

Code Interpreter is an intelligent document generation tool that can create, edit, and format professional documents based on your natural language requests. It automatically applies Texas Capital Bank templates to ensure brand consistency across all generated content.

## How to Use Code Interpreter

Simply describe what you want to create in your message. For example: "Create a PowerPoint presentation on Introduction to Machine Learning with 5 slides"

Ranger will automatically detect the need for document generation and activate Code Interpreter to create your requested content.

![Using Code Interpreter](/images/useofcode_interpreter.png)

Once generated, Ranger will display the document in the chat. Click on the document to automatically download it.

## Texas Capital Templated File Downloads

Code Interpreter generates documents in Texas Capital Bank's official templates for:

- **PowerPoints (.pptx)** - Professional presentations with TCB branding
- **Word Documents (.docx)** - Formatted documents with TCB styling
- **PDF (.pdf)** - Polished reports and documents
- **Excel (.xlsx)** - Spreadsheets and data analysis files

All generated files are automatically formatted with Texas Capital Bank's brand standards, ensuring consistency and professionalism in every document.

## Example Use Cases

- Create client presentation decks
- Generate quarterly reports
- Build financial analysis spreadsheets
- Draft proposal documents
- Create training materials
- Develop project documentation
