## Attachments

Ranger allows users to attach files directly in the chat using the attachment icon in the message box. The attachment menu provides two options that enable Ranger to interact with users by processing and managing content from images and documents.

### Overview

The attachment feature in Ranger provides two distinct upload options:
1. Upload Image
2. Upload Documents

<Callout type="info">
  **General Limitations:**
  - Maximum of 20 files per upload
  - Each file size must not exceed 10 MB
  - Documents uploaded in a chat thread are accessible only in that thread
</Callout>

![Attach Files](/images/attachfilesv.png)

### Upload Image

The Upload Image option allows users to upload images directly into the chat. Users can share visual content easily for reference or discussion.

**Steps:**
1. Click the attachment icon in the chat input
2. Select **Upload Image**
3. Choose an image file to upload

**Example Use Cases:**
- Sharing screenshots
- Displaying diagrams or charts
- Visual content reference for discussions

<Callout type="tip">
  Uploaded images are displayed directly in the chat thread, making it easy to discuss visual content with Ranger.
</Callout>

### Upload Documents

Selecting Upload Documents automatically activates the File Search tool. Users can see the File Search activated indicator in the message box after the tools panel symbol.

**Steps:**
1. Click the attachment icon in the chat input
2. Select **Upload Documents**
3. Choose the files to upload. The File Search tool is automatically activated, allowing Ranger to search within the uploaded files

**Supported File Types:**
- Microsoft Office: .docx, .pptx, .xlsx
- Database: .sql
- PDF: .pdf
- CSV: .csv
- Text/Markdown: .txt, .md
- HTML: .html
- JSON/YAML: .json, .yaml
- Source Code: .py, .js, and other code files

**Example Use Cases:**
- Searching through uploaded PDFs, Word documents, or spreadsheets in the current thread
- Quickly finding specific content in large documents uploaded for reference
- Analyzing multiple documents simultaneously with search capabilities

<Callout type="tip">
  When the File Search tool is activated, you can ask natural language questions about the content of your uploaded files.
</Callout>

### Document Management

Managing your uploaded files is simple through the account section:

- All uploaded files for any option are visible in **Account â†’ My Files** in the left panel
- Users can delete uploaded files from My Files
- Files cannot be downloaded
- Files uploaded in one chat thread cannot be accessed in other threads
- Users can continue to chat through documents uploaded within that thread

![Available Files](/images/availablefiles.png)

<Callout type="warning">
  Remember that documents are thread-specific - if you need to reference a document in multiple conversations, you'll need to upload it in each thread separately.
</Callout>

### Best Practices for Attaching Files

**For Image Uploads:**
- Ensure images are clear and have good resolution
- Crop unnecessary parts before uploading
- Use descriptive questions when asking about image content

**For Document Uploads:**
- Use specific questions referencing content you expect to find
- Utilize natural language queries instead of keywords
- Reference specific sections or page numbers when possible
- Upload related documents together for comparative analysis

### Summary

Ranger's file attachment features provide flexible options for incorporating various file types into your conversations:

- **Upload Image:** Share visual content directly in the chat
- **Upload Documents:** Enable powerful search capabilities across uploaded documents

By utilizing these file attachment options effectively, you can enhance your interactions with Ranger through rich content sharing and document analysis capabilities.
