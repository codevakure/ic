# Speech Settings

Speech settings enable voice interaction with Ranger, offering both simple and advanced modes for hands-free operation.

**When to Use**: When you prefer voice interaction or need hands-free operation of Ranger.

## Simple Mode

Simple Mode provides basic voice input and output capabilities using your browser's built-in speech features.

![Speech Simple Mode](/images/speechsimple.png)

### Speech to Text Engine
**Engine**: Browser-based speech recognition

**When to use**: Configure for basic voice input when working on your own device.

Features:
- Uses your browser's native speech recognition
- No additional setup required
- Works offline in most browsers
- Good for basic dictation and commands

### Language Selection
**Language**: Select your preferred spoken language

**When to use**: Adjust to match your spoken language for better recognition accuracy.

Considerations:
- Choose your primary speaking language
- Match your accent/dialect when available
- Consider your team's common language for shared sessions

### Text to Speech Engine
**Engine**: Browser-based text-to-speech

**When to use**: Use for basic voice output without additional cloud services.

Features:
- Uses browser's native TTS capabilities
- No internet connection required for basic voices
- Immediate response without delays
- Good for basic read-aloud functionality

## Advanced Mode

Advanced Mode provides enhanced speech capabilities with additional features and customization options.

![Speech Advanced Mode](/images/speechadv.png)

### Auto Transcribe Audio
**Auto Transcribe Audio**: Automatically convert spoken input to text

**When to use**: Enable for automatic transcription of meetings, dictation, or continuous speech input.

Benefits:
- Hands-free conversation flow
- Automatic punctuation and formatting
- Real-time speech-to-text conversion
- Suitable for longer dictation sessions

### Auto Send Settings
**Auto Send Text**: Configure automatic message sending with customizable delay

**When to use**: Adjust for smoother voice interactions (use -1 to disable automatic sending).

Options:
- **Immediate**: Send as soon as speech ends
- **Delayed**: Wait specified seconds before sending
- **Manual**: Require manual confirmation (-1 setting)
- **Smart**: Detect natural speech pauses

### Autoplay Features
**Autoplay Latest Message**: Automatically read new responses aloud

**When to use**: Enable for hands-free operation or accessibility needs.

Scenarios:
- Multitasking while getting information
- Accessibility support for visual impairments
- Driving or hands-busy situations
- Learning through auditory processing

### Cloud-Based Voices
**Use Cloud-Based Voices**: Access higher quality speech synthesis

**When to use**: Enable for higher quality speech output when internet connection is available.

Advantages:
- More natural-sounding voices
- Better pronunciation of complex terms
- Multiple voice personalities and accents
- Improved emotional expression

### Audio Playback Control
**Audio Playback Rate**: Control speech speed (Default = 1.0)

**When to use**: 
- Increase for faster information consumption
- Decrease for better comprehension of complex content
- Adjust based on your listening preferences

Rate guidelines:
- **0.5-0.8**: Slower for learning or complex content
- **1.0**: Normal conversational speed
- **1.2-1.5**: Faster for familiar content
- **1.5+**: Speed reading equivalent

### TTS Cache Management
**Enable Cache TTS**: Store generated speech for improved performance

**When to use**: Enable to improve performance when using text-to-speech frequently.

Benefits:
- Faster replay of previously generated speech
- Reduced bandwidth usage
- Consistent voice quality for repeated content
- Offline playback of cached content

<Callout type="tip">
  Speech features are particularly valuable for accessibility purposes or when you need hands-free operation, such as while multitasking or for users with mobility limitations.
</Callout>

## Best Practices for Speech Settings

### Getting Started
- Begin with Simple Mode to test basic functionality
- Ensure your microphone and speakers are properly configured
- Test in a quiet environment initially
- Gradually move to Advanced Mode as you become comfortable

### Optimization Tips
- Use a quality headset microphone for best recognition
- Speak clearly and at a consistent pace
- Minimize background noise during voice input
- Position microphone consistently for better accuracy

### Advanced Usage
- Enable cloud voices for professional presentations
- Use auto-transcribe for meeting notes and documentation
- Adjust playback speed based on content complexity
- Cache frequently used responses for offline access

### Accessibility Considerations
- Configure autoplay for visual accessibility support
- Use slower playback rates for better comprehension
- Enable auto-send for mobility-limited users
- Set up consistent voice commands for routine tasks

### Troubleshooting
- Check browser permissions for microphone access
- Verify internet connection for cloud-based features
- Clear TTS cache if experiencing voice quality issues
- Test different voices if current selection is unclear

<Callout type="info">
  Start with browser-based speech features to ensure compatibility, then gradually enable cloud-based options for enhanced quality and functionality.
</Callout>
