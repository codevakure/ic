# Chat Settings

Chat settings allow you to customize how messages display and how you interact with the chat interface.

**When to Use**: When you need to customize how messages display and how you interact with the chat interface.

![Chat Settings](/images/chat.png)

## Display Options

### Message Font Size
Control the size of text in your conversations for optimal readability.

Options: **Small** / **Medium** / **Large**

**When to use**: 
- Increase for better readability or visual accessibility
- Decrease to fit more content on screen
- Adjust based on your display size and resolution

### Chat Direction
Set the text direction for your conversations.

Options: **Left-to-right (LTR)** / **Right-to-left (RTL)**

**When to use**: Adjust based on your preferred language direction or regional text conventions.

## Interaction Preferences

### Message Sending
**Press Enter to send messages**: Control how the Enter key behaves in the message input.

- **Enable**: Quick single-line messages (Enter sends, Shift+Enter for new line)
- **Disable**: Easier multi-line editing (Enter creates new line, manual send)

**When to use**: Enable for quick single-line messages, disable for easier multi-line editing.

### Workspace Layout
**Maximize chat space**: Optimize screen real estate for conversation focus.

- **Enable**: Hide non-essential UI elements for distraction-free chatting
- **Disable**: Keep all interface elements visible

**When to use**: Enable when you need to focus on conversation content without distractions.

### Reasoning Display
**Open Thinking Dropdowns by Default**: Automatically show Ranger's reasoning process.

- **Enable**: Always see how Ranger approaches complex problems
- **Disable**: Keep reasoning collapsed to save space

**When to use**: Enable to automatically see Ranger's reasoning process for complex queries.

### Tool Call Thoughts
**Show Human-Readable Tool Descriptions**: Display friendly descriptions instead of technical tool names.

- **Enable**: See "Searching the web for information" instead of "tavily_search_results_json"
- **Disable**: Show technical function names and raw tool calls

**When to use**: Enable for a more intuitive understanding of what Ranger is doing behind the scenes. Similar to Claude's reasoning display, this shows thought processes in expandable cards with human-readable descriptions.

### Code Display
**Always show code when using code interpreter**: Control code visibility during programming tasks.

- **Enable**: See all code execution and development steps
- **Disable**: Show only results unless specifically requested

**When to use**: Enable when working with programming tasks to see all code execution.

## Advanced Features

### Mathematical Content
**Parse LaTeX in messages**: Enable mathematical equation rendering.

- **Enable**: Display mathematical formulas and equations properly
- **Disable**: Show LaTeX as plain text (better performance)

**When to use**: Enable when working with mathematical content, disable to improve performance.

<Callout type="tip">
  If you frequently work with complex mathematical formulas, enabling LaTeX parsing will allow for clearer representation of equations in your conversations.
</Callout>

### Draft Management
**Save drafts locally**: Preserve in-progress messages.

- **Enable**: Auto-save message drafts as you type
- **Disable**: Drafts are lost if you navigate away

**When to use**: Enable to preserve in-progress messages if you need to leave and return later.

### Navigation Aids
**Scroll to the end button**: Add a quick navigation button for long conversations.

- **Enable**: Show button to jump to latest message
- **Disable**: Use manual scrolling only

**When to use**: Enable for easier navigation in long conversations.

### Visual Indicators
**Save badges state**: Maintain visual status indicators across sessions.

- **Enable**: Remember badge states between sessions
- **Disable**: Reset badges each session

**When to use**: Enable to maintain visual indicators across sessions.

### Conversation Flow
**Show follow-up suggestions**: Display AI-generated conversation prompts.

- **Enable**: Get guided suggestions for continuing conversations
- **Disable**: Clean interface without suggestions

**When to use**: Enable for guided conversation flow, disable for cleaner interface.

## Conversation Controls

### Model Switching
**Enable switching endpoints mid-conversation**: Allow model changes during active chats.

- **Enable**: Switch between different AI models within the same conversation
- **Disable**: Lock to initial model choice

**When to use**: Enable when working with complex tasks that might require different models.

### Forking Options
**Use default fork option**: Streamline conversation branching.

- **Enable**: Use preset fork settings automatically
- **Disable**: Choose fork options each time

**When to use**: Enable to streamline creating conversation branches.

**Start fork from target message by default**: Control where conversation branches begin.

- **Enable**: New branches start from selected message
- **Disable**: Branches start from conversation end

**When to use**: Enable to create conversation branches starting from specific points.

## Best Practices for Chat Settings

### Display Optimization
- Use larger fonts on high-resolution displays or for extended reading sessions
- Adjust chat direction based on your primary language or regional preferences
- Maximize chat space when focusing on complex conversations

### Interaction Efficiency
- Enable Enter-to-send for quick back-and-forth conversations
- Show thinking dropdowns when learning how AI approaches problems
- Enable code display when working on programming projects

### Advanced Feature Usage
- Enable LaTeX parsing for mathematical, scientific, or technical discussions
- Save drafts when working on complex, multi-part queries
- Use follow-up suggestions to explore topics more thoroughly

### Conversation Management
- Enable mid-conversation model switching for complex problem-solving
- Use default fork options to quickly explore alternative approaches
- Start forks from target messages to explore specific decision points

<Callout type="info">
  Keyboard shortcuts can dramatically improve your efficiency when working with Ranger, especially during intensive work sessions requiring frequent model or prompt switching.
</Callout>
