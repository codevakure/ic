# Commands Settings

Commands settings help you optimize your workflow with keyboard shortcuts and quick command access.

**When to Use**: When you want to optimize your workflow with keyboard shortcuts and command access.

![Commands Settings](/images/commands.png)

## Keyboard Shortcuts

Keyboard shortcuts can dramatically improve your efficiency when working with Ranger, especially during intensive work sessions requiring frequent model or prompt switching.

### Model Switching Shortcut
**Toggle "@" for switching endpoints/models**: Enable quick model switching with the @ symbol.

- **Enable**: Type "@" to quickly switch between different AI models
- **Disable**: Use standard interface methods for model selection

**When to use**: Enable for quick model switching during technical tasks that require different AI capabilities.

**How it works**:
- Type "@" in the message box
- Select from available models/endpoints
- Continue typing your message with the new model

### Multi-Response Settings
**Toggle "+" for multi-response settings**: Access multiple response options quickly.

- **Enable**: Type "+" to configure multiple response generation
- **Disable**: Use standard interface for multi-response features

**When to use**: Enable when you frequently need multiple response options for comparison or different perspectives.

**How it works**:
- Type "+" in the message box
- Configure multiple response parameters
- Generate several responses simultaneously

### Prompt Selection
**Toggle "/" for prompt selection via keyboard**: Quick access to prompt templates and saved queries.

- **Enable**: Type "/" to access prompt library via keyboard
- **Disable**: Use interface buttons for prompt selection

**When to use**: Enable for quick access to prompt templates and saved queries.

**How it works**:
- Type "/" in the message box
- Browse available prompts and templates
- Select and customize prompts without leaving the keyboard

### Document Selection
**Toggle "#" for document selection via keyboard**: Quick access to previously uploaded documents.

- **Enable**: Type "#" to access your uploaded documents library via keyboard
- **Disable**: Use interface buttons for document selection

**When to use**: Enable for quick access to reference previously uploaded files without re-uploading.

**How it works**:
- Type "#" in the message box
- Browse your available uploaded documents
- Select documents to reference in your conversation

## Command Usage Examples

### Model Switching with "@"
```
@thinking-model What's the best approach to solve this algorithm?
@instant-model Quick summary of this document
@research-model Detailed analysis of market trends
```

### Multi-Response with "+"
```
+3 different perspectives on this business strategy
+compare Give me technical and non-technical explanations
```

### Prompt Selection with "/"
```
/email-template Draft a professional follow-up email
/report-format Create a quarterly business report
/code-review Analyze this code for improvements
```

### Document Selection with "#"
```
#quarterly-report What are the key metrics from this report?
#meeting-notes Summarize the action items from this document
#research-paper Explain the methodology used in this study
```
