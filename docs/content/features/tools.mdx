# Tools

Ranger provides powerful tools in the chat interface that allow users to extend functionality and interact with files, the web, or generate content dynamically.

## Overview

Ranger gives you access to specialized features that enhance your workflow:

- Available tools include:
  - [File Search (RAG-enabled)](/features/tools/file-search)
  - [Web Search](/features/tools/web-search)
  - [Artifacts](/features/tools/artifacts)
  - [Code Interpreter](/features/tools/code-interpreter)
  - [SharePoint & Office 365 Suite](/features/tools/sharepoint-office365)
  - [Fetch URL](/features/tools/fetch-url)

**How Tool Selection Works:**
- **Automatic Selection:** Most tools (File Search, Artifacts, Code Interpreter) are automatically activated based on your prompt. Ranger intelligently detects which tool is needed for your request.
- **Manual Selection Required:** Web Search and Microsoft 365 tools must be manually selected before use.

**Deactivating a Tool:**
- To unselect a tool, simply click the tool name in the chat interface
- This deactivates the tool and removes it from the active chat view

<Callout type="info">
  Tools are thread-specific. Activation and uploaded content only affect the current chat thread.
</Callout>

## Available Tools

### [File Search (RAG-enabled)](/features/tools/file-search)
Automatically activated for document analysis with RAG technology. Perfect for searching through uploaded documents using natural language queries.

### [Web Search](/features/tools/web-search)
**Manual activation required.** Provides real-time web search capabilities for current information, market trends, and regulatory updates.

### [Artifacts](/features/tools/artifacts)
Automatically activated for generating React components, HTML code, and Mermaid diagrams. Great for prototyping and visualization.

### [Code Interpreter](/features/tools/code-interpreter)
Automatically activated for generating professional documents and presentations in Texas Capital Bank's branded templates.

### [SharePoint & Office 365 Suite](/features/tools/sharepoint-office365)
**Manual activation required.** Seamless integration with Microsoft 365 ecosystem including SharePoint, OneDrive, Outlook, and Teams.

### [Fetch URL](/features/tools/fetch-url)
**Manual activation required.** Retrieve and analyze content from specific web pages or documents using URLs.

## Combining Tools for Enhanced Workflow

Ranger's tools can be used in combination to create powerful workflows:

**Example Workflow: Data Analysis**
1. Upload spreadsheets using File Search
2. Ask questions about the data using natural language
3. Generate visualizations with Artifacts
4. Save templates of common charts using Artifacts

**Example Workflow: Research Project**
1. Enable Web Search to gather the latest information
2. Upload reference documents with File Search
3. Create diagrams to visualize findings using Artifacts
4. Generate summary visualizations to present conclusions

**Example Workflow: Client Presentation**
1. Use Code Interpreter to generate a professional PowerPoint presentation
2. Upload supporting data with File Search
3. Create visualizations to display key metrics
4. Download the final TCB-branded presentation

**Example Workflow: Office 365 Integration**
1. Use SharePoint to find the latest project documents
2. Query OneDrive for supporting files
3. Send summary emails to team members through Outlook
4. Reference Teams conversations for additional context

## Best Practices for Tools

- **Be Specific:** When using tools, clearly state what you're looking for to get the most relevant results
- **Use the Right Tool:** Select the appropriate tool for your task rather than trying to adapt a less suitable tool
- **Combine with Attach Files:** Use the file attachment feature alongside tools for the most comprehensive workflow
- **Iterate and Refine:** Don't hesitate to refine your requests if the initial results aren't what you expected

Ranger's Tools provide powerful extensions to enhance your productivity. Click on any tool above to learn more about its specific features and usage instructions.
